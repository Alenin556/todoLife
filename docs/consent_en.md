# Data Processing Consent (Template)

Date: 2026-04-28

This is a **template** used when the App includes optional features that require consent (analytics, crash reporting, advertising, cloud sync, etc.).

## 1) Default behavior

By default, the App does **not** send your content (tasks, notes, finance data) to a server.

## 2) Optional consent (examples)

You can separately enable:

- **Crash reports**: technical crash information to improve stability.
- **Anonymous analytics**: aggregated usage events to improve UX.

## 3) Withdrawal

You can withdraw consent at any time in Settings → **Privacy** by disabling the respective toggles.

## 4) Links

- Privacy Policy: `docs/privacy_policy_en.md` (or the public URL in the store listing)
