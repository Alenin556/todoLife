# Согласие на обработку данных (Consent) — шаблон

Дата: 2026-04-28

Этот документ — **шаблон**. Он применяется, если в приложении включаются функции, требующие согласия (аналитика, crash‑репорты, реклама, облачная синхронизация и т. п.).

## 1) Что по умолчанию

По умолчанию приложение **не отправляет** ваш контент (задачи, заметки, финансовые данные) на сервер.

## 2) На что даётся согласие (опционально)

Вы можете отдельно включить:

- **Отчёты о сбоях** (crash‑репорты): техническая информация о сбое для исправления ошибок (например, версия приложения, модель устройства/ОС, стек ошибки).
- **Анонимная аналитика**: агрегированные события использования для улучшения UX (например, открытие экранов/нажатия кнопок/создание записи — без содержимого).

Важно: при включении этих функций **не отправляется** ваш контент: **текст задач/заметок**, **финансовые суммы**, **содержимое календарных событий** и другие пользовательские записи.

Дополнительно: вы можете вручную сформировать **баг‑репорт** в настройках (копируется в буфер обмена) и отправить его разработчику. Такой отчёт содержит техническую информацию и последние логи приложения, без содержимого ваших записей.

## 3) Как отозвать согласие

Согласие можно в любой момент отозвать в настройках: раздел **Конфиденциальность** → переключатели функций.
