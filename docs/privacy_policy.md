# Политика конфиденциальности todoLife

Дата вступления в силу: 2026-04-28

Эта Политика конфиденциальности описывает, какие данные обрабатывает приложение **todoLife** (далее — «Приложение»), для каких целей, где они хранятся и какие у пользователя есть права.

## 2) Какие данные обрабатываются

### 2.1 Данные, которые вы вводите в приложении (контент)

Приложение может хранить на вашем устройстве:

- задачи (текст, статус выполнения)
- календарные события (название, дата/время, заметки, параметры напоминаний)
- финансовые данные раздела «Средства» (суммы/проценты/категории, сохранённые бюджеты)
- настройки приложения (например, тема)

### 2.2 Технические данные (опционально)

Если в Приложении будет включена отправка диагностических отчётов (по вашему согласию), могут отправляться:

- техническая информация о сбоях (стектрейсы)
- данные об устройстве/ОС (версия, модель), необходимые для диагностики

Мы **не** намерены отправлять содержимое ваших задач/заметок/финансов в диагностические отчёты.

## 3) Цели обработки

Мы обрабатываем данные для:

- работы функций Приложения (задачи, календарь, финансы)
- обеспечения стабильности и исправления ошибок (если вы включили диагностику)

## 4) Где хранятся данные

По умолчанию данные хранятся **локально на вашем устройстве**.

Если в будущем появится облачная синхронизация/аккаунт, Политика будет обновлена, и будет указано:

- какие данные отправляются на сервер
- где расположен сервер/провайдер
- сроки хранения и меры защиты

## 5) Передача третьим лицам

По умолчанию Приложение **не передаёт** ваши данные третьим лицам.

Если будут подключены сторонние сервисы (аналитика, crash‑репорты, реклама), это будет:

- явно обозначено в настройках
- включаться **только по вашему согласию**
- отражено в этой Политике (с перечнем сервисов и ссылками на их политики)

## 6) Разрешения (permissions)

Приложение может запрашивать разрешения, необходимые для функций (например, уведомления).

Вы можете управлять разрешениями через настройки ОС.

## 7) Сроки хранения

Локальные данные хранятся, пока вы их не удалите или не выполните «Сброс данных».

Диагностические данные (если включены) могут храниться у провайдера ограниченное время (обычно до 90 дней).

## 8) Безопасность

Мы применяем разумные меры для защиты данных, включая:

- минимизацию данных
- ограничение логирования
- защищённое локальное хранение (шифрование/KeyStore/Keychain — в зависимости от платформы)

## 9) Ваши права и управление данными

В Приложении предусмотрены возможности:

- удалить отдельные записи (задачи/события/бюджеты)
- выполнить **полный сброс данных** (удалить всё локальное хранилище)
- экспортировать данные (если функция доступна)

## 10) Дети

Приложение не предназначено специально для детей. Если вы считаете, что ребёнок предоставил данные, свяжитесь с нами по контактам выше.

## 11) Изменения политики

Мы можем обновлять Политику. Новая версия публикуется вместе с датой вступления в силу.
