# todoLife v2.2.3

`todoLife` — кроссплатформенное приложение для задач, финансов и календаря с акцентом на дисциплину: ежедневные чек‑листы, долгосрочные цели, календарные события и напоминания, плюс инструменты финансового планирования.

## Ключевые функции

- **Задачи**:
  - **Сегодня**: чекбоксы, статистика “Выполнено X из Y”, добавление/редактирование/удаление. При смене дня **незавершённые задачи переносятся на сегодня**.
  - **Долгосрочные**: создаются из того же экрана, живут до завершения/удаления, можно задавать дедлайн.
  - **Переключатель Сегодня / Долгосрочные**: сегмент‑переключатель отображается только если в приложении есть долгосрочные задачи.
- **Дедлайн долгосрочной задачи (опционально)**: при наличии дедлайна задача отображается в календаре на выбранный день.
- **Календарь**:
  - месячный календарь + список событий на день
  - создание/редактирование событий с заметкой
  - **выбор времени** (12‑часовой формат AM/PM, минуты шагом 5)
  - **напоминания**: уведомления приходят **за N минут до начала** (только если указано время и выбран “Напомнить за …”)
- **Финансы** (один раздел, вкладки):
  - **Средства**: ввод суммы, распределение по категориям (прямой ввод сумм + процентные “подсказки”), сохранение/удаление бюджетов, “Сбросить значения”.
  - **Калькулятор вклада**: сложные проценты, срок в месяцах/годах, итоговая сумма и прибыль.
  - **План вкладов**: расчёт ежемесячного взноса, чтобы к сроку накопить цель (с учётом уже накопленной суммы и опциональной годовой доходности).
- Маршрут: **`/finance`**. Старые пути `/finance/salary` и `/finance/deposit` редиректятся в **`/finance`**.
- **Главная**: фотоподложка + минималистичная композиция + мини‑календарь недели + сводка событий и мотивационные цитаты (автосмена).

## Что нового в v2.2.0 (безопасность + улучшения)

### Безопасность и приватность

- **Защищённое хранение данных**: задачи/календарь/финансы хранятся в **secure storage** (Android KeyStore / iOS Keychain / macOS Keychain) с миграцией со старого хранилища.
- **Раздел “Конфиденциальность”** в настройках:
  - просмотр документов: **Privacy Policy / Terms / Consent**
  - **Экспорт данных** (TXT в буфер обмена, на языке приложения)
  - **Сброс всех данных** (полное удаление локальных данных)
- **Язык приложения**: переключатель **RU / EN** (документы открываются на выбранном языке).
- **Блокировка приложения (Вариант B)**:
  - **PIN и/или биометрия** при возврате в приложение
  - **автоблокировка** (сразу или через N секунд)
  - **скрытие содержимого** при сворачивании (в переключателе приложений)
  - **запрет скриншотов/записи экрана на Android** (FLAG_SECURE, включаемо в настройках)

### Финансы

- **Распределение по суммам** корректно отображается:
  - в итоговом блоке при ручном вводе
  - в **“Сохранённых бюджетах”** (учитываются ручные суммы и пользовательские категории)

### Задачи и уведомления

- **Перенос ежедневных задач**: незавершённые daily задачи переходят на новый день.
- **Уведомление при первом входе за день**: список задач на сегодня + кнопка “Далее” для закрытия (Android).

### Документы для маркетплейсов

- Добавлены файлы в `docs/`:
  - `privacy_policy.md`, `terms.md`, `consent.md`
  - (и их английские версии — для публикации в сторе)

## Что нового в v2.2.0

- **Конфиденциальность**:
  - новый экран **Настройки → Конфиденциальность**
  - просмотр документов: **Политика конфиденциальности / Условия / Согласие (шаблон)** (markdown‑ассеты в `docs/`)
  - **Экспорт данных**: копирование JSON в буфер обмена (по инициативе пользователя)
  - **Сброс данных**: полное удаление локальных данных (задачи/календарь/финансы)
- **Хранение данных (безопаснее)**:
  - “чувствительные” данные переносятся в **secure storage** (KeyStore/Keychain) там, где доступно
  - добавлена миграция старых данных из `SharedPreferences` (best‑effort)
  - загрузка данных стала устойчивой к битым/устаревшим payload (не крашится при `jsonDecode`)
- **Ежедневные задачи**:
  - при смене дня незавершённые задачи переносятся на сегодня, выполненные не переносятся
  - показ “сводки задач” один раз в день при первом запуске (локальное уведомление)

## Безопасность релиза v2.2 (чеклист)

- **Данные пользователя**:
  - убедиться, что приложение **ничего не отправляет** наружу без явного согласия
  - проверить, что **экспорт** — только локально (clipboard), без сетевых запросов
  - проверить “Сброс данных”: удаляет `SharedPreferences` и secure storage
- **Уведомления**:
  - не включать в уведомления чувствительные данные (финансы/заметки); оставлять только минимум текста
  - проверить permissions (Android 13+): запрос только при необходимости
- **Сборка release**:
  - собрать APK с обфускацией и раздельными символами:
    - `flutter build apk --release --obfuscate --split-debug-info=build/symbols/v2.2.0`
  - хранить `build/symbols/v2.2.0` локально (не коммитить) — пригодится для расшифровки стектрейсов
  - проверить `flutter analyze` и widget‑тесты перед публикацией
