# todoLife v2.0.0

`todoLife` — кроссплатформенное приложение для задач, финансов и календаря.

## Ключевые функции

- **Задачи на день**: чекбоксы, статистика “Выполнено X из Y”, добавление/редактирование/удаление. Список автоматически очищается при смене даты.
- **Долгосрочные задачи**: отдельный список без автоочистки.
- **Дедлайн для долгосрочных задач (опционально)**: можно указать дату; при наличии дедлайна задача фиксируется в календаре на этот день.
- **Распределение ЗП**: ввод суммы, распределение по системным категориям (проценты) + пользовательские категории (фикс. суммы), контроль превышения остатка, сохранение бюджетов, отображение остатка.
- **Депозитный калькулятор**: сложные проценты, срок в месяцах/годах, итоговая сумма и прибыль.
- **Календарь**: месяц, подсветка дней с событиями, события по дню, добавление/редактирование события (время/заметка/напоминание — опционально).
- **Главная**: фотоподложка + минималистичная композиция + мини-календарь недели + сводка событий и мотивационные цитаты (автосмена).

## Что нового в v2.0.0 (UI/UX Release)

- **Новый визуальный стиль**: премиальная тёмная тема + “песочная” светлая тема (Material 3, единые отступы/радиусы).
- **Главная страница 1:1 по референсу**: фото-фон, минималистичная типографика, мини-календарь недели с точками событий и быстрым переходом в день.
- **Мотивация на главной**: цитаты показываются прямо на Home и **автоматически меняются каждые 30 секунд**.
- **Новая навигация**:
  - **Mobile**: нижняя навигация (вкладки) вместо Drawer, состояние вкладок сохраняется.
  - **Web**: верхняя навигация (как в референсе) + переключатель темы.
- **Настройки**: отдельный экран (переключение Light/Dark).
- **Локализация RU**: русский календарь/пикеры дат.
- **Иконки приложения**: новые SVG + автогенерация PNG, обновлены Android/iOS launcher icons.
- **Web Deploy**: автоматический деплой на GitHub Pages через GitHub Actions (SPA fallback + relative base-href).

## Технологии

- **Flutter / Dart**
- **go_router** (StatefulShellRoute для вкладок + web top-nav)
- **shared_preferences** (локальное хранение)
- **intl** (даты/форматирование)

## Запуск

Установите Flutter и проверьте окружение:

```bash
flutter doctor
```

Затем:

```bash
flutter pub get
flutter run
```

## Сборки (Release)

### Android APK

```bash
flutter build apk --release
```

Файл: `build/app/outputs/flutter-apk/app-release.apk`

### Windows

```bash
flutter build windows --release
```

Файл: `build/windows/x64/runner/Release/todolife.exe`

### Web

GitHub Pages разворачивается автоматически при пуше в `main` через workflow:
`\.github/workflows/deploy_web.yml`

## Структура

- `lib/ui/screens/tasks/` — задачи
- `lib/ui/screens/finance/` — финансы и депозит
- `lib/ui/screens/calendar/` — календарь
- `lib/ui/screens/home/` — главный экран
- `lib/ui/screens/settings/` — настройки
- `lib/services/user_storage.dart` — локальное хранилище
- `lib/app_state.dart` — состояние приложения
