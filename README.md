# todoLife v2.3

`todoLife` — кроссплатформенное приложение для задач, финансов и календаря с акцентом на дисциплину: ежедневные чек‑листы, долгосрочные цели, календарные события и напоминания, плюс инструменты финансового планирования.

## Ключевые функции

- **Задачи**:
  - **Сегодня**: чекбоксы, статистика “Выполнено X из Y”, добавление/редактирование/удаление. Список автоматически очищается при смене даты.
  - **Долгосрочные**: создаются из того же экрана, живут до завершения/удаления, можно задавать дедлайн.
  - **Переключатель Сегодня / Долгосрочные**: сегмент‑переключатель отображается только если в приложении есть долгосрочные задачи.
- **Дедлайн долгосрочной задачи (опционально)**: при наличии дедлайна задача отображается в календаре на выбранный день.
- **Календарь**:
  - месячный календарь + список событий на день
  - создание/редактирование событий с заметкой
  - **выбор времени** (12‑часовой формат AM/PM, минуты шагом 5)
  - **напоминания**: уведомления приходят **за N минут до начала** (только если указано время и выбран “Напомнить за …”)
- **Распределение ЗП**: ввод суммы, распределение по категориям (прямой ввод сумм + процентные “подсказки”), сохранение/удаление бюджетов, “Сбросить значения”.
- **Депозитный калькулятор**: сложные проценты, срок в месяцах/годах, итоговая сумма и прибыль.
- **Главная**: фотоподложка + минималистичная композиция + мини‑календарь недели + сводка событий и мотивационные цитаты (автосмена).

## Что нового в v2.3 (последние улучшения)

- **Задачи**: объединены в один раздел `/tasks` — внутри переключатель **Сегодня / Долгосрочные** + FAB‑меню создания (обычная задача / долгосрочная с иконкой часов).
- **Календарь**: возвращён выбор времени “реалистичнее” — **12h AM/PM** и минуты **шагом 5**.
- **Android‑уведомления**: добавлены локальные уведомления календаря (звук + вибра) **только если задан `startTime`** и выбрано “Напомнить за …” (при редактировании/удалении событие пересинхронизируется).
- **Цитаты**: расширены до **~100** строк, автосмена каждые **30 секунд** сохранена.
- **Иконка приложения**: обновлена генерация — центральный **“глаз”** + перегенерация iOS/Android ассетов.
- **Тесты**: расширены widget‑тесты под новый UX задач и отображение времени в дневном просмотре календаря.

## Что было добавлено ранее (v2.0 UI/UX Release)

- **Новый визуальный стиль**: премиальная тёмная тема + “песочная” светлая тема (Material 3, единые отступы/радиусы).
- **Главная страница**: фото‑фон, минималистичная типографика, мини‑календарь недели с точками событий и быстрым переходом в день.
- **Мотивация на главной**: цитаты показываются прямо на Home и **автоматически меняются каждые 30 секунд**.
- **Навигация**:
  - **Mobile**: нижняя навигация (вкладки) вместо Drawer, состояние вкладок сохраняется.
  - **Web**: верхняя навигация (как в референсе) + переключатель темы.
- **Настройки**: отдельный экран (переключение Light/Dark).
- **Локализация RU**: русский календарь/пикеры дат.
- **Иконки приложения**: новые SVG + автогенерация PNG, обновлены Android/iOS launcher icons.
- **Web Deploy**: автоматический деплой на GitHub Pages через GitHub Actions (SPA fallback + relative base-href).

## Технологии

- **Flutter / Dart**
- **go_router** (StatefulShellRoute для вкладок + web top-nav)
- **shared_preferences** (локальное хранение)
- **intl** (даты/форматирование)
- **flutter_local_notifications** + **timezone** (Android напоминания)

## Запуск

Установите Flutter и проверьте окружение:

```bash
flutter doctor
```

Затем:

```bash
flutter pub get
flutter run
```

## Сборки (Release)

### Android APK

```bash
flutter build apk --release
```

Файл: `build/app/outputs/flutter-apk/app-release.apk`

### Windows

```bash
flutter build windows --release
```

Файл: `build/windows/x64/runner/Release/todolife.exe`

### Web

GitHub Pages разворачивается автоматически при пуше в `main` через workflow:
`\.github/workflows/deploy_web.yml`

## Структура

- `lib/ui/screens/tasks/` — задачи
- `lib/ui/screens/finance/` — финансы и депозит
- `lib/ui/screens/calendar/` — календарь
- `lib/ui/screens/home/` — главный экран
- `lib/ui/screens/settings/` — настройки
- `lib/services/user_storage.dart` — локальное хранилище
- `lib/app_state.dart` — состояние приложения
